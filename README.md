  ### _Demo for `for-each` loop in java_
  ```java
  String names[] = {"bhaskar","apple","Banana","Bingo"};
  for(Stirng name : names){
   System.out.print(name);
  }
  
  ```
  
  ### output:
  `bhasker
   apple
   Banana
   Bingo
  `
